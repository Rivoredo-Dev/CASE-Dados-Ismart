# 📊 Case Ismart - Análise de Dados Educacionais

Este repositório contém o desenvolvimento de um estudo de caso focado no programa de bolsas do **Ismart**. O objetivo principal é analisar o desempenho acadêmico e o perfil socioeconômico dos alunos para gerar insights sobre o impacto do programa.

### 🚀 Objetivos do Projeto
* **Limpeza de Dados:** Tratamento da base bruta para garantir a integridade das análises.
* **Análise Exploratória (EDA):** Identificação de padrões de sucesso acadêmico e desafios enfrentados pelos bolsistas.
* **Métricas de Performance:** Acompanhamento de indicadores educacionais ao longo do tempo.
* **Geração de Insights:** Apoio à tomada de decisão baseada em dados para melhoria do programa.

### 🛠️ Estrutura do Repositório
* `Base_Estag_Dados.csv`: Conjunto de dados contendo informações detalhadas sobre os estudantes.
* `Descricao_Case_Dados.pdf`: Documentação com as diretrizes e desafios propostos para a análise.
* `Case_Ismart.docx`: Relatório final ou orientações estratégicas sobre o desenvolvimento do case.

### 🧰 Tecnologias Sugeridas
* **Excel / Google Sheets:** Para triagem inicial e tabelas dinâmicas.
* **Python (Pandas/Matplotlib):** Para análises estatísticas e visualizações complexas.
* **Power BI / Tableau:** Para criação de dashboards de acompanhamento.

### 🏁 Como Utilizar
1. Faça o download da base de dados disponível em formato `.csv`.
2. Consulte o arquivo de descrição para entender o contexto das variáveis e os problemas de negócio a serem resolvidos.
3. Execute as análises focando na relação entre o perfil do bolsista e seu desempenho final.
